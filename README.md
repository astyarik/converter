# Конвертер

RUS

Простой скрипт для конвертации в консоли.

Может:
Конвертировать Цельсии в Кельвины и обратно  
Рубли в Гривны и обратно (фиксированный курс за 30 декабря 2025 года)
Километры в метры и обратно

Гайд по использованию:
1. python conv_ru.py  
2. Вводишь цифру в меню. 
3. Вводишь число  
4. Получаешь результат конвертации  
5. Для выхода в меню используйте "7"

Курс валют:
1 руб. = 0.5397 грн.
1 грн. = 1.85 руб.

# Converter

ENG

A simple console script for conversions.

Can:
Convert Celsius to Kelvin and back  
Convert Rubles to Hryvnias and back (fixed rate as of December 30, 2025)
Convert Kilometrs to Metrs and back

How to use:
1. python conv_en.py  
2. Enter a number from the menu  
3. Enter the number to convert  
4. Get the conversion result  
5. To exit, choose "7" from the menu

Exchange rates:
1 RUB = 0.5397 UAH  
1 UAH = 1.85 RUB
